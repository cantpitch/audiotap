Choose Convert from a TAP file

  • Choose to a WAV file if you want to create a WAV file. It can be later played back and recorded to a real tape. This option is not available if the file libaudiofile-0.dll is missing from the directory where audiotap.exe is.
  • Choose to a sound if you want to play back the sound right away. If a tape recorder's input is connected to the sound card output, a tape can be recorded directly. This option is not available if the file libportaudio-2.dll is missing from the directory where audiotap.exe is.
Inverted waveform: some sound cards invert the signal, that is they turn negative values of the waveform into positive ones and vice versa. If your sound card is one of those, your Commodore will be unable to load the tape. In these cases, check this box. If your C64 does not load the tape you recorded, try toggling this box and start again.
Advanced options

Then press OK. You will be asked to choose the input TAP (or DMP) file, and, in case you choose conversionto a WAV file, the output WAV file. Wait until the end, or press Cancel to stop immediately.
